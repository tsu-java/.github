# рЃЏрЃЮрЃњрЃћрЃАрЃљрЃџрЃЏрЃћрЃЉрЃўрЃЌ ­ЪЉІ

TSU Java рЃљрЃарЃўрЃА рЃњрЃўрЃбрЃ░рЃљрЃЉрЃўрЃА рЃЮрЃарЃњрЃљрЃюрЃўрЃќрЃљрЃфрЃўрЃБрЃџрЃў рЃљрЃюрЃњрЃљрЃарЃўрЃерЃў, рЃарЃЮрЃЏрЃћрЃџрЃерЃўрЃф рЃЌрЃЉрЃўрЃџрЃўрЃАрЃўрЃА рЃАрЃљрЃ«рЃћрЃџрЃЏрЃгрЃўрЃцрЃЮ рЃБрЃюрЃўрЃЋрЃћрЃарЃАрЃўрЃбрЃћрЃбрЃўрЃА рЃЎрЃЮрЃЏрЃърЃўрЃБрЃбрЃћрЃарЃБрЃџрЃў рЃЏрЃћрЃфрЃюрЃўрЃћрЃарЃћрЃЉрЃўрЃА рЃЉрЃљрЃЎрЃљрЃџрЃљрЃЋрЃарЃўрЃА рЃАрЃбрЃБрЃЊрЃћрЃюрЃбрЃћрЃЉрЃўрЃА (рЃЊрЃљ рЃЏрЃљрЃЌ рЃерЃЮрЃарЃўрЃА рЃЕрЃћрЃЏрЃўрЃф) рЃ»рЃљрЃЋрЃљрЃќрЃћ рЃЊрЃљрЃгрЃћрЃарЃўрЃџрЃў рЃърЃарЃЮрЃћрЃЦрЃбрЃћрЃЉрЃў (рЃАрЃљрЃфрЃљрЃЋрЃћрЃЉрЃў) рЃљрЃарЃўрЃА рЃњрЃљрЃюрЃЌрЃљрЃЋрЃАрЃћрЃЉрЃБрЃџрЃў.

рЃљрЃЦ рЃарЃљ рЃЌрЃЦрЃЏрЃљ рЃБрЃюрЃЊрЃљ рЃДрЃЋрЃћрЃџрЃљ рЃърЃарЃЮрЃћрЃЦрЃбрЃў рЃљрЃарЃљрЃљ рЃњрЃљрЃюрЃЌрЃљрЃЋрЃАрЃћрЃЉрЃБрЃџрЃў. рЃЏрЃ«рЃЮрЃџрЃЮрЃЊ рЃарЃЕрЃћрЃБрЃџрЃў рЃърЃарЃЮрЃћрЃЦрЃбрЃћрЃЉрЃў рЃљрЃарЃўрЃА рЃЊрЃљ рЃўрЃЦрЃюрЃћрЃЉрЃљ рЃњрЃљрЃюрЃЌрЃљрЃЋрЃАрЃћрЃЉрЃБрЃџрЃў, рЃарЃЮрЃЏрЃџрЃћрЃЉрЃќрЃћ рЃЏрЃБрЃерЃљрЃЮрЃЉрЃљрЃф рЃюрЃћрЃЉрЃўрЃАрЃЏрЃўрЃћрЃа рЃљрЃЊрЃљрЃЏрЃўрЃљрЃюрЃА рЃерЃћрЃћрЃФрЃџрЃћрЃЉрЃљ.
